# Instructions for Manually Setting Git Bash options

1. Open git bash terminal
2. Right click title bar select options

## Options

### Looks

- Cursor
  - Underscore
  - Blinking

### Text

- Font
  - Lucida Console, 11pt

### Keys

- Default settings

### Mouse

- Right mouse button
  - Paste

### Selection

- Default

### Window

- Default size
  - Columns: 115
  - Rows: 50

### Terminal

- Default settings
